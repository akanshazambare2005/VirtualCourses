🎓 AI-Powered Learning Management System (LMS)

A full-stack AI-enabled Learning Management System developed using the MERN Stack, featuring secure authentication, online payments, and an admin dashboard for course management.

📌 Project Overview

Designed and developed a scalable LMS web application with role-based access for students and administrators

Implemented secure authentication and payment workflows suitable for real-world deployment

Integrated AI features to enhance user learning experience and content interaction

🚀 Key Features

User Authentication

Google OAuth 2.0 login

JWT-based authorization and protected routes

Course Management

Browse, enroll, and access courses

Admin-controlled course creation, updates, and deletion

Payment Integration

Razorpay integration for secure course purchases

Payment verification and order management

Admin Panel

Role-based admin access

User, course, and payment management dashboard

AI Capabilities

AI-powered learning assistant

Smart content recommendations and responses
